This tool support for:

- Get java script error on console log
- Run link checker with support on http or https
- Get any conten in source page by search in regular expression

How to use:
- Install Java SDK( version >1.7 ) and set JAVA_HOME path, can check by command java -version in command line
- Downadload MiniTool.jar, GetScriptError_lib folder, TestRun folder and put all in same folder
- Click run MiniTool.jar and enjoy automation for get javascript error, run link checher and get content of page source
